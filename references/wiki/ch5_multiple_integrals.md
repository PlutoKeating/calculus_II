# 阶段五：重积分（二重·三重）

> 🟢 **来自资料**：真题必考——极坐标二重积分、交换积分次序、变量替换（雅可比）、对称性化简、三重积分的柱/球坐标（2020-21、22-23 期末，2024 期中）。
> 🟡 **AI补充**：公式为标准知识。

## 一、二重积分
**基本思路**：∫∫_D f dA，先定积分区域 D，再选坐标系。
- **直角坐标**：画出 D，定内外层上下限（先积一个变量看它的上下界是另一变量的函数）。
- **交换积分次序**：先把给定累次积分还原成区域 D 的不等式，重画 D，再按另一顺序写限（真题 ∫₁/₂¹∫_{x³}^{x} 型需分段）。
- **极坐标**（圆/圆盘/含 x²+y² 时首选）：x=r cosθ, y=r sinθ, **dA = r dr dθ**。
  - 典型边界：x²+y²=2x → r=2cosθ（圆过原点）。
- **一般变量替换**：u=u(x,y),v=v(x,y)，dA = |J| du dv，J=∂(x,y)/∂(u,v)（雅可比行列式）。真题：u=x, v=x+y 把 e^{x/(x+y)} 化成可积形式。

## 二、对称性化简（送分技巧）
- 若 D 关于某轴/直线对称，且被积函数关于对应变量为**奇函数** → 积分 = 0。
- 若关于 y=x 对称，可把 f(x,y) 与 f(y,x) 相加取平均（真题 cos²(x²+y)+sin²(x+y²) → 平均后 =1 → 结果 = 面积）。

## 三、三重积分
∫∫∫_Ω f dV，选坐标系：
- **直角坐标**：dV=dx dy dz；「先一后二」或「先二后一（切片法）」。
- **柱坐标**（含 x²+y²、圆柱/圆锥区域）：x=r cosθ,y=r sinθ,z=z，**dV = r dz dr dθ**。
- **球坐标**（含 x²+y²+z²、球区域）：x=ρsinφcosθ, y=ρsinφsinθ, z=ρcosφ，**dV = ρ²sinφ dρ dφ dθ**。
  - 例：∫∫∫(x²+y²+z²)dV 于单位球 = ∫₀^{2π}∫₀^π∫₀¹ ρ²·ρ²sinφ dρdφdθ = 4π/5。

## 四、标准答题模板
1. **看区域形状 + 被积函数**选坐标系（圆→极/柱，球→球，一般→换元）。
2. 画区域、定限（三重先定 z 的上下界，再投影到 xy 定二重限）。
3. 换元写对**面积/体积元**（极坐标 r、球坐标 ρ²sinφ，别漏）。
4. 先查**对称性**能否直接化简或置零。

## 五、易错点
- ⚠️ 极坐标漏乘 r、球坐标漏乘 ρ²sinφ（最常见失分）。
- ⚠️ 交换次序前**务必画区域图**，否则限容易写错。
- ⚠️ 变量替换的雅可比取**绝对值**。
- ⚠️ 用对称性置零前，要**同时**满足「区域对称 + 函数奇性」。

## 六、3 分钟速记
「**圆用极坐标乘 r，球用球坐标乘 ρ²sinφ；换次序先画图；对称奇函数直接归零。**」

# 阶段六：向量场与曲线曲面积分（格林/高斯/斯托克斯）

> 🟢 **来自资料**：期末压轴集中区——散度/旋度、两类曲线积分、路径无关与保守场、格林公式、两类曲面积分、高斯散度定理、斯托克斯定理，真题反复考（2020-21、22-23 期末每题都涉及）。
> 🟡 **AI补充**：公式为标准知识。

## 一、场的运算
- 梯度（标量场→向量场）：∇f=(f_x,f_y,f_z)。
- **散度**（向量场→标量）：div F = ∇·F = ∂M/∂x+∂N/∂y+∂P/∂z。（真题 F=e^{xy}i+cos(xy)j+xz²k → div = ye^{xy}−x sin(xy)+2xz）
- **旋度**（向量场→向量）：curl F = ∇×F = |i j k; ∂x ∂y ∂z; M N P|。

## 二、曲线积分
- **第一类（对弧长，标量）** ∫_C f ds：参数化 x=x(t),y=y(t)，ds=√(x'²+y'²)dt。
  - 若在 C 上 √(x²+y²)=常数 a，则 ∫_C √(x²+y²)ds = a·(弧长)（真题 x²+y²=a² → 2πa²）。
- **第二类（对坐标，向量做功）** ∫_C F·dr = ∫_C M dx+N dy+P dz：① 参数化代入；② 环路用格林；③ 路径无关时用势函数。

## 三、路径无关与保守场（高频）
平面上 ∫ M dx+N dy 路径无关 ⟺ **∂M/∂y = ∂N/∂x**（单连通域内）⟺ 存在势函数 u，du=M dx+N dy。
- **算法**：先验 ∂M/∂y=∂N/∂x；再求势函数 u（对 M 关于 x 积分，再用 N 定待定函数）；积分值 = u(终点)−u(起点)。空间同理，条件 curl F=0。

## 四、格林公式（平面环路）
∮_C M dx+N dy = ∫∫_D (∂N/∂x − ∂M/∂y) dA（C 为 D 的正向边界，逆时针）。
- 真题：∮(2xy−2y)dx+(x²−4x)dy 于 x²+y²=9 → ∫∫(−2)dA = −2·9π = −18π。
- **奇点处理**（分母含 (x−a)²+(y−b)²）：在奇点外挖一个小圆，用格林 + 小圆上直接参数化（真题 ∮[(x−1)dy−y dx]/[(x−1)²+y²]=2π）。

## 五、曲面积分
- **第一类（对面积）** ∫∫_Σ f dS：投影 dS=√(1+z_x²+z_y²)dA，或参数化 dS=|r_u×r_v|du dv。
  - 对称性：球面上奇函数积分 = 0（真题 (x+y³+sin z)/(1+z⁴) 于球面 → 0）。
- **第二类（通量）** ∫∫_Σ F·n dS = ∫∫ M dydz+N dzdx+P dxdy。

## 六、三大定理（何时用）
- **高斯散度定理**（闭曲面通量）：∯_Σ F·n dS = ∫∫∫_Ω div F dV。用于**闭曲面**求通量。
  - 真题：F=(x³+…, y³+…, 3z−x³) 于实心圆柱 → ∫∫∫(3x²+3y²+3)dV = 108π。
  - 曲面**不闭合**时：补一块面凑成闭曲面，用高斯算总量再减补面。
- **斯托克斯定理**（空间环路 ↔ 旋度通量）：∮_C F·dr = ∫∫_Σ (curl F)·n dS。边界好算用左式，曲面好参数化用右式。
- **平面格林**是斯托克斯在平面的特例。

## 七、选择策略（压轴题决策树）
1. 闭曲面求通量 → **高斯**。
2. 空间闭曲线做功 → **斯托克斯**（或先看是否保守，保守则只需端点）。
3. 平面闭曲线 → **格林**；有奇点先挖洞。
4. 曲面不闭 → 补面用高斯；曲线不闭且非保守 → 参数化。

## 八、易错点
- ⚠️ 高斯/格林要求**闭合 + 正向**；方向反了差一个负号，缺口要补面。
- ⚠️ 保守场判据是 curl F=0（平面 ∂M/∂y=∂N/∂x）**且区域单连通**；有奇点时不能直接用。
- ⚠️ 第二类积分要注意**侧/法向**方向（上侧/外侧）。
- ⚠️ dS 与 dA 的关系别漏 √(1+z_x²+z_y²)。

## 九、3 分钟速记

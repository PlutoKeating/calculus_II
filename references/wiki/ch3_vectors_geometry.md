# 阶段三：向量与空间解析几何

> 🟢 **来自资料**：真题常以填空/小题出现——叉积、过三点/过点且垂直的平面方程、两直线夹角、旋转曲面、空间曲线的切线（2020-21 期末、2024 期中均考）。
> 🟡 **AI补充**：公式为标准知识。

## 一、向量基本运算
- **点积** a·b = |a||b|cosθ = a₁b₁+a₂b₂+a₃b₃ → 求**夹角/投影/判垂直**（a·b=0 ⟺ 垂直）。
- **叉积** a×b：结果是**向量**，垂直于 a,b；|a×b|=|a||b|sinθ=平行四边形面积。行列式记法：
  a×b = |i j k; a₁ a₂ a₃; b₁ b₂ b₃|。
- 小技巧：(a−b)×(a+b) = a×a − b×a + a×b − b×b = 2(a×b)。

## 二、平面与直线
- **平面**：点 P₀ + 法向量 n=(A,B,C) → A(x−x₀)+B(y−y₀)+C(z−z₀)=0。
  - 过三点：n = (P₂−P₁)×(P₃−P₁)。
  - 过点且垂直于某平面/含某方向：n 由「叉积」把已知的两个方向叉出来。
- **直线**：点 + 方向向量 v → 对称式 (x−x₀)/v₁=(y−y₀)/v₂=(z−z₀)/v₃。
  - 两平面交线的方向 v = n₁×n₂。
  - **两直线夹角**：cosθ = |v₁·v₂|/(|v₁||v₂|)（取绝对值保证是锐角）。
- **平行平面间距离**：Ax+By+Cz=d₁ 与 =d₂ → d = |d₁−d₂|/√(A²+B²+C²)。

## 三、空间曲线与曲面
- **空间曲线** r(t)=x(t)i+y(t)j+z(t)k：
  - 切向量 = r'(t)；某点切线对称式用 r(t₀) 作点、r'(t₀) 作方向。
- **旋转曲面**：yz 平面曲线 z=ky 绕 z 轴旋转 → 把 y 换成 ±√(x²+y²)：z=±k√(x²+y²)。（绕哪个轴，就把「另外两个变量」用到该轴的距离替换）
- 常见二次曲面：椭球、椭圆抛物面、双曲面、锥面（了解形状即可）。

## 四、标准答题模板
- **平面方程**：定 n（叉积或垂直条件）→ 点法式 → 化一般式。
- **切线对称式**：算 r'(t₀)（方向）→ 算 r(t₀)（点）→ 写 (x−x₀)/…=(y−y₀)/…=(z−z₀)/…。
- **夹角**：取方向向量点积 / 模长积，**加绝对值**。

## 五、易错点
- ⚠️ 叉积**不可交换**：a×b = −b×a；分量行列式别抄错顺序。
- ⚠️ 夹角忘了取绝对值 → 可能算出钝角。
- ⚠️ 旋转曲面替换时分清绕哪个轴、保留哪个变量。

## 六、3 分钟速记
「**点积求角、叉积求法向；平面点法式、直线点向式；旋转换根号、夹角带绝对。**」

# 阶段二：幂级数与泰勒级数

> 🟢 **来自资料**：真题极高频——「求收敛域 + 和函数」几乎每套卷必有（2020-21 期末 Σ x²ⁿ⁺²/((n+1)(2n+1))、2024 期中 Σ(2n-1)x²ⁿ⁻²/2ⁿ）；「把函数展成幂级数」也常考（arctan、ln((1+x)/(1-x))）。
> 🟡 **AI补充**：定义与公式为标准知识，记号以老师为准。

## 一、大白话隐喻
幂级数就是**「无穷次多项式」**：c₀+c₁x+c₂x²+…。它只在某段 x 上「说得通」（收敛），这段区间就是**收敛域**；在收敛域内它 = 某个熟悉的函数，那个函数就是**和函数**。

## 二、收敛半径 / 收敛域（标准流程）
对 Σ aₙ(x−x₀)ⁿ：
1. **求半径 R**：ρ = lim|aₙ₊₁/aₙ|（或 lim ⁿ√|aₙ|），R = 1/ρ。得开区间 |x−x₀|<R **内部绝对收敛**。
2. **单独测两端点** x=x₀±R：代入得数项级数，用第一章判别法逐个判（这是最容易丢分的一步）。
3. 合并写出收敛域（可能是 (a,b)、[a,b)、(a,b]、[a,b]）。
- 对「缺项」级数（如只有 x²ⁿ）用 **比值法直接对整项** lim|uₙ₊₁/uₙ|<1 解 x 更稳。

## 三、求和函数四板斧（收敛域内可逐项微/积）
1. **逐项求导 / 逐项积分**把系数里的 (n+1)、1/(2n+1) 等「消化」成已知几何级数。
2. 已知基石：1/(1−x)=Σxⁿ (|x|<1)。
3. 由导数/积分回推：求导后凑成 1/(1−x) 型 → 求出 S'(x) → 再积分 + 定初值 S(x₀) 定常数。
4. 常见结果：Σxⁿ/n = −ln(1−x)；Σ(-1)ⁿx²ⁿ⁺¹/(2n+1)=arctan x；Σx²ⁿ⁺¹/(2n+1)=½ln((1+x)/(1-x))。

## 四、泰勒 / 麦克劳林级数
- 定义：f(x)=Σ f⁽ⁿ⁾(x₀)/n! ·(x−x₀)ⁿ。麦克劳林即 x₀=0。
- **五个必背展开**（|收敛域|）：
  - eˣ = Σ xⁿ/n!（全体 ℝ）
  - sin x = Σ(-1)ⁿ x²ⁿ⁺¹/(2n+1)!；cos x = Σ(-1)ⁿ x²ⁿ/(2n)!
  - 1/(1−x)=Σxⁿ (|x|<1)；ln(1+x)=Σ(-1)ⁿ⁻¹ xⁿ/n (−1<x≤1)；(1+x)ᵅ 二项级数
- **套路展开**：把目标函数先求导得到「已知展开」的形式，展开后**再逐项积分**回去（arctan、ln 类专用）。
- **求高阶导数** f⁽ⁿ⁾(0)：先展成幂级数，则 f⁽ⁿ⁾(0)=n!·(xⁿ 的系数)。奇偶性缺项时对应导数=0（如 2024 期中 x/(1−x²) 只含奇次幂 → f⁽²⁰²⁴⁾(0)=0）。
- **在 x₀≠0 处展开**（如 1/(x²−x−6) 在 x=4）：先**部分分式**，再把每个 1/(x−a) 写成 1/((x₀−a)+(x−x₀)) 提出常数配几何级数。

## 五、易错点
- ⚠️ 求出 R 后**必须**单独判两端点，别直接写开区间。
- ⚠️ 逐项微/积只在**开区间内**成立；端点的和另需取极限（如 x→1⁻）验证。
- ⚠️ 求和后要**回代 x=0 或已知点**核对常数项，避免积分常数错误。
- ⚠️ 部分分式后，几何级数各自的收敛范围不同，最终收敛域取**交集**。

## 六、3 分钟速记
「**比值求 R，端点单独判；求和先微积、凑 1/(1−x)，积回定常数；泰勒背五式、缺项导数零。**」

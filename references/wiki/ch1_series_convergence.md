# 阶段一：数列与常数项级数（收敛判别）

> 🟢 **来自资料**：本章「必考」定位来自真题——历年期中/期末每套卷都有 1~2 道「判别敛散 / 绝对·条件收敛」的题（如 22-23 期末 Σ(-1)ⁿ sin n /(n√n)、2020-21 期末 Σ(-1)ⁿ⁺¹/√n）。
> 🟡 **AI补充**：下面的定义、判别法和解题模板是标准微积分知识，可能与你老师的措辞/记号略有出入，以老师课件为准。

## 一、大白话隐喻
把「级数收敛」想成**往一个杯子里一次次倒水**：每次倒的量是 aₙ。如果倒到最后水位稳定在某个高度 → **收敛**；如果水一直往上涨、溢出 → **发散**。判别法就是「不用真的倒完，就预测水位会不会稳住」的各种快捷判断。

## 二、判敛「工具箱」（考场按顺序试）

**第 0 步（必做）项趋于 0 检验**
- 若 lim(n→∞) aₙ ≠ 0（或不存在）→ **级数发散**（一票否决）。
- 注意：aₙ→0 **不能**推出收敛（如调和级数 Σ1/n，项趋 0 却发散）。

**① p-级数（记忆基准）**：Σ 1/nᵖ ⟺ p>1 收敛，p≤1 发散。（p=1 即调和级数，发散）

**② 几何级数**：Σ arⁿ ⟺ |r|<1 收敛，和 = 首项/(1−r)。

**③ 积分判别法**：aₙ=f(n)，f 连续、正、递减 → Σaₙ 与 ∫₁^∞ f(x)dx 同敛散。

**④ 比较判别法**：0≤aₙ≤bₙ，则 Σbₙ 收敛 ⇒ Σaₙ 收敛；Σaₙ 发散 ⇒ Σbₙ 发散。

**⑤ 极限比较法**（最常用）：aₙ,bₙ>0，lim aₙ/bₙ = L。
- 0<L<∞ → 同敛散（挑一个已知的 bₙ 做「标尺」，通常用 p-级数）。
- 典型：用等价无穷小把复杂项换成 1/nᵖ，如 1−cos(1/nᵏ) ~ ½·(1/nᵏ)² = 1/(2n²ᵏ)。

**⑥ 比值法（ratio）**：lim |aₙ₊₁/aₙ| = ρ。ρ<1 收敛，ρ>1 发散，ρ=1 失效。（含 n! / rⁿ 时首选）

**⑦ 根值法（root）**：lim ⁿ√|aₙ| = ρ。判据同比值法。（含 (…)ⁿ 时首选）

## 三、交错级数与绝对/条件收敛（高频）

**莱布尼茨（交错级数）判别法**：Σ(-1)ⁿ bₙ，若 bₙ>0 且 **单调递减** 且 bₙ→0 → 收敛。

**绝对收敛 vs 条件收敛（答题三段式）**：
1. 先看 Σ|aₙ|：收敛 → 原级数 **绝对收敛**（结束）。
2. 若 Σ|aₙ| 发散，再看原级数（用交错级数判别法）：收敛 → **条件收敛**；发散 → **发散**。

## 四、标准答题模板（考场照抄框架）
- **判绝对/条件**：① 写出 Σ|aₙ|；② 对 |aₙ| 放缩或比较（常配 p-级数）→ 判 Σ|aₙ| 敛散；③ 若绝对收敛写结论；若不绝对收敛，再用莱布尼茨判原级数 → 写「条件收敛」。
- **一般判敛**：先查 aₙ→0；再挑工具（含 n!/rⁿ→比值；含 (·)ⁿ→根值；有理式/等价无穷小→极限比较）。

## 五、易错点
- ⚠️ aₙ→0 只是**必要**条件，绝不能当收敛的证据。
- ⚠️ 莱布尼茨要**同时**满足「递减 + 趋 0」；只说趋 0 不够。
- ⚠️ 比较法要求**正项**；交错/变号级数先取绝对值再比较。
- ⚠️ 极限比较选标尺时，先用等价无穷小估计 aₙ 的「阶」（约等于 1/n 的几次方）。

## 六、3 分钟速记
「**先验零、再挑法；p 记 1、比几何；n! 用比值，幂次用根值；变号先绝对，绝对不成看交错。**」

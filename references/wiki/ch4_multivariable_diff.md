# 阶段四：多元函数微分学

> 🟢 **来自资料**：本章是期中/期末的「主力得分区」——极限不存在证明、可微性证明、链式法则（含二阶混合偏导）、隐函数求导、方向导数/梯度、极值与拉格朗日乘数法，真题反复出现（22-23、2024 期中，2020-21、22-23 期末）。
> 🟡 **AI补充**：定义与判定为标准知识。

## 一、多元极限「不存在」的证明（高频 6 分题）
思路：**找两条路径，极限值不同 → 不存在。**
- 常用路径：沿坐标轴（如 x=0,y=0,z=t）、沿直线 y=kx、沿 x=y=z=t 等。
- 例（22-23 期末）：lim xyz/(x³+y³+z³) 沿 x=y=z=t 得 1/3，沿 x=y=0,z=t 得 0 → 不存在。

## 二、偏导数、可微、连续的关系（易考概念题）
- 偏导存在 ⇏ 可微；可微 ⇒ 偏导存在且连续。
- **可微定义**：Δz − [f_x·Δx + f_y·Δy] = o(ρ)，ρ=√(Δx²+Δy²)。
- **证不可微套路**（真题原题 f=xy/(x²+y²)，(0,0) 处补 0）：
  1. 用定义算 f_x(0,0)、f_y(0,0)（多为 0）；
  2. 考察极限 [Δz − f_xΔx − f_yΔy]/ρ 沿 y=kΔx，若结果**依赖 k**（≠0）→ 极限不存在 → 不可微。

## 三、链式法则（复合函数偏导，含抽象 f）
- 记号：f₁=∂f/∂(第1个中间变量)，f₁₁、f₁₂ 为二阶偏导。
- **标准步骤**：① 写清中间变量 u,v 对 x,y 的偏导；② 一阶：z_x = f₁·u_x + f₂·v_x；③ 求二阶时，**f₁、f₂ 本身仍是 (u,v) 的函数**，要再套一次链式法则；④ 用 f₁₂=f₂₁（二阶连续）合并。
- 例（真题）z=f(u,x,y), u=xeʸ：z_x = eʸf₁+f₂ → ∂²z/∂x∂y = eʸf₁ + x e²ʸ f₁₁ + eʸ f₁₃ + x eʸ f₂₁ + f₂₃。

## 四、隐函数求导
- 单方程 F(x,y,z)=0 定 z=f(x,y)：z_x = −F_x/F_z，z_y = −F_y/F_z。
- 也可**两边同时对 x 求偏导**（把 z 看成 x,y 的函数）再解出 z_x（考场更直观）。
- 抽象方程 G(x/z, y/z)=0：链式 + 隐函数，常出「求 x·z_x + y·z_y」的对称结论（真题答案 = z）。

## 五、方向导数与梯度
- 梯度 ∇f = (f_x, f_y, f_z)。
- 方向导数 D_u f = ∇f · u（u 为**单位**方向向量）。
- **最大方向导数 = |∇f|**，方向沿 ∇f；最小 = −|∇f|。（真题：f=x²+2y² 在 (0,1) 最大方向导数 = |(0,4)| = 4）
- 切平面：曲面 F(x,y,z)=0 在 P₀ 的切平面法向量 = ∇F(P₀)。

## 六、极值与最值
- **无约束极值（二阶判别）**：解 f_x=f_y=0 得驻点；令 A=f_xx,B=f_xy,C=f_yy，D=AC−B²。
  - D>0 且 A>0 → 极小；D>0 且 A<0 → 极大；D<0 → 非极值（鞍点）；D=0 → 失效。
- **条件极值（拉格朗日乘数法）**：L = f + λ₁g₁ + λ₂g₂（每个约束一个 λ）；令所有偏导=0 + 约束方程，解出候选点，代回 f 比大小。
  - 例（真题）u=x−2y+2z 在球 x²+y²+z²=1 上：max=3，min=−3（=±|(1,−2,2)|）。

## 七、易错点
- ⚠️ 证极限不存在只需**两条路径不同**；但要证「存在=L」需夹逼/极坐标，别混。
- ⚠️ 求二阶偏导时，f₁ 还是复合函数，**别忘了继续对它链式求导**。
- ⚠️ 方向导数的 u 必须**单位化**。
- ⚠️ 拉格朗日别漏「约束方程本身」；最后要**代回目标函数比较**才知谁是 max/min。

## 八、3 分钟速记
「**两路不同证不存在；链式记 f₁ 仍是函数；隐函数 −F_x/F_z；梯度模是最大方向导；D=AC−B² 判极值，有约束就上拉格朗日。**」
